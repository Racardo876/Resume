# Resume Website (GitHub Pages)

This folder contains a single-page, modern resume website:

- `index.html` — main resume page
- `styles.css` — modern/clean styling + print-friendly rules
- `js/theme-toggle.js` — light/dark theme toggle (saved in `localStorage`)
- `assets/resume.pdf` — the PDF used by the **Download PDF** link
- `assets/headshot.jpg` — optional profile photo (fallbacks to initials if missing)

## Deploy to GitHub Pages

1. Create a Git repository from this folder (or upload these files into an existing one).
2. Push to GitHub.
3. In your GitHub repo settings, go to **Pages** and set the source to **Deploy from a branch**.
4. Select the branch you pushed (commonly `main`) and the `/` root directory.
5. The published URL should serve the site immediately.
